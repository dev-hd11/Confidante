## Dependencies :

2. Django

## Installation :
1. Install Runner 1.0
2. Create a `.env` file in the Confidante/confidante folder and add this:
   ```python
   SECRET_KEY='key'
   DEBUG=False
   ```

   Replace `key` with your key.
3. Run Runner 1.0
4. Go to [http://localhost:8000](http://localhost:8000).
