## Installation :
1. Download the latest release.
2. Navigate to NEXTVerse/nextverse
3. Run the `install.bat` or type <br>
   ```cmd
   npm i
   ```
   in the terminal
4. Run the `startserver.bat` or type <br>
   ```cmd
   npm run dev

   ```
   in the terminal

5. Go to [http://localhost:3000](http://localhost:3000) on your device.
